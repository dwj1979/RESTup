
%serverVersion% - RESTful сервер консольных приложений. WebDAV интерфейс.

Принцип действия интерфейса: 
 - выбираете папку сервиса;
 - копируете во вложенную папку "%inFilesFolder%" исходные файлы;
 - возвращаете результат из вложенной папки "%outFilesFolder%".
   Для некоторых WebDAV клиентов необходимо принудительно обновить её содержимое.
   В случае ошибки сервиса в папке создается файл "Oops! Internal Server Error".

Совокупный размер Ваших файлов ограничен %sessionQuota% МиБ, а время хранения - %sessionTimeout% мин. 

В приведенном ниже списке папки сервисов отмечены знаком "+". 
Для них указаны допустимые расширения (типы) исходных файлов,
(*) означает "любой файл", включая создание подкаталогов.
Папки, отмеченные "-", служат целям группировки и доступны только для чтения.
